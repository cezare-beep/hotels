1. Клонирование репозитория
git clone https://github.com/cezare-beep/hotels.git

2. Установка зависимостей
pip install flask flask-socketio requests websocket-client

3. Запуск сервера
python api.py
Сервер запустится на http://127.0.0.1:5000

4. Запуск клиента
python gui.py
